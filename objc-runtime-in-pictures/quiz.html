Quiz

Armed with this knowledge, here is an example project. Assume that you are only linking against the mulle-objc-runtime, which doesn’t provide any classes.

@interface A
@property char   *x;
+ (instancetype) new;
@end
@implementation A
+ (instancetype) new
{
   return( mulle_objc_infraclass_alloc_instance( self));
}
- (char *) x  {  return( (char *) __FUNCTION__); }
@end

@protocol B
@optional
- (char *) x;
@end
@interface B
- (char *) x;
@end
@implementation B
- (char *) x  {  return( (char *) __FUNCTION__); }
@end


@interface C : A < B >
@end
@implementation C
@end

1. How many metaclasses will exist at runtime ?

2. How many methods does infraclass C have in its methodlists ?

3. What is the result of [[C new] x] ?

4. What is the result of [C x] ?
